# ORA-01840 #

![error](https:/images/2023-04-23-1차프로젝트오류/inselet(사진)/ORA-01840.png)

ORA-01840: input value not long enough 
for date format
(날짜형식에 맞지 않음)

<br/><br/>


# 원인 # 

날짜형식에 맞지 않는 값 1을 입력했기 때문에 발생한 오류.

<br/>

# 해결 #

19990101 or 990101 로 입력.

<br/><br/>
